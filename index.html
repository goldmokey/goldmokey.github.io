Githug小游戏玩转git

发表于 2017-04-30

Githug

一个命令行的git闯关小游戏。把平常操作过程中遇到的一些场景都进行分解案例化，变成了一个个关卡。目前总计55关，有趣又好玩的git。

项目传送门

1 Install

当然安装过程可能没这么顺利。 爬坑专用 1 爬坑专用2

2 Githug Help

开始游戏之前，不了解规则怎么战胜boss。最有用的命令之一

，会指引你前进的道路。

  • githug help [TASK] 帮助
  • githug hint 指点
  • githug levels 展示
  • githug play 初始化
  • githug reset 重启关卡
  • githug test 测试
