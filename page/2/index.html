一个命令行的git闯关小游戏。把平常操作过程中遇到的一些场景都进行分解案例化，变成了一个个关卡。目前总计55关，有趣又好玩的git。

项目传送门

1 Install

当然安装过程可能没这么顺利。 爬坑专用 1 爬坑专用2

开始游戏之前，不了解规则怎么战胜boss。最有用的命令之一

，会指引你前进的道路。

  • githug help [TASK] 帮助
  • githug hint 指点
  • githug levels 展示
  • githug play 初始化
  • githug reset 重启关卡
  • githug test 测试

Git基础篇-命令行

发表于 2017-04-28 | 分类于 Git学习

  • init
    初始化一个仓库

  • config
    全局或本地配置用户名和邮箱

  • add
    添加文件

  • commit
    提交文件

  • clone
    克隆一个仓库，默认文件夹名就是仓库名

  • clone_to_folder
    克隆一个仓库，并指定文件夹

  • ignore
    编辑.gitignore文件,这样提交时可以忽略某些或某类文件，当然使用强大的正则表达式

  • include
    包含某些文件，使用同ignore

  • status
    查看所有untracked的文件
